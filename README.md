# ECU_Scaling_Maps
Logiciel de mise a l'echelle des cartographies moteur. Permet le redimentionnement par interpolation les cartes. 
Exemple passer d'une 4x4 a une 6X6 ou 3X12 en conservant les valeurs et proportions.
